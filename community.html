Get involved

Ask Question and Discuss

If you have any question or want to discuss some idea or problem?

You can reach us on various channels:

  • Twitter: @SBuildOrg
  • IRC: SBuild-org/sbuild on Gitter (preferred), #sbuild on Freenode
  • Mailing lists:
    • sbuild-dev: Discussions about the development of SBuild
    • sbuild-user: for SBuild users

Report Bugs and Feature Requests

You found a bug? Browse our issue tracker on GitHub and file an issue if your bug was not reported before.

Please note, that most Plugins have their own issue trackers.

Commit your Code

SBuild is open source and hosted on GitHub. You can fork it and issue a pull request. For larger changes, we advise to discuss your planned change on the

  • mailing list before.

    Follow us

    Follow us on Twitter or per RSS Feed to get useful information.

    Mention us with the hashtag #SBuild, star and watch our GitHub repositories. If your write some article about SBuild, please let us know.
